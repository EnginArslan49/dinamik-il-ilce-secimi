2. Veritabanı Bağlantısının Yapılandırılması

Projedeki Models klasörü içerisinde yer alan DbIlIlceSecimiContext.cs dosyasındaki bağlantı ayarlarını kendi ortamınıza göre düzenleyin.

DEĞİŞTİRİLMESİ GEREKEN KOD:
optionsBuilder.UseSqlServer("Server=<localhost, . veya .\\SQLEXPRESS>;Database=DB_il_ilceSecimi;User Id=<sqlServerDogrulamaKullaniciAdiniz>;Password=<sqlServerDogrulamaSifreniz>;TrustServerCertificate=True;");

# Server, User Id ve Password alanlarını kendi SQL Server yapılandırmanıza uygun şekilde güncelleyin.
